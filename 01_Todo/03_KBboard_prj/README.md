# 📌 Vue 게시판 목록

## 📋 프로젝트 개요

Vue.js를 활용하여 간단한 게시판 목록을 구현한 프로젝트입니다. 사용자는 게시글 목록을 클릭하여 상세 내용을 확인할 수 있으며, 닫기 버튼을 통해 게시글을 닫을 수 있습니다.

## ✨ 주요 기능

- 게시글 목록 표시
- 게시글 클릭 시 상세 내용 확인
- 닫기 버튼을 통한 게시글 숨기기
- Vue.js `ref()`를 활용한 반응형 데이터 관리

## 🛠 사용된 기술

- **Frontend**: Vue.js 3, HTML, CSS
